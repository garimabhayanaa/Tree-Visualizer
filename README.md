Tree-Visualizer

📌 Description

Tree-Visualizer is a web-based tool that allows users to input an array of numbers and visualize them in different tree-based data structures. Based on the selected option, the tool sorts and organizes the data into:

Binary Tree

Binary Search Tree (BST)

Min Heap

Max Heap

This interactive visualization helps users understand how different tree structures work and how data is arranged in each.

🚀 Features

✅ Accepts an array of numbers as input
✅ Visualizes data as Binary Tree, BST, Min Heap, and Max Heap
✅ Interactive and intuitive UI for better understanding
✅ Dynamic sorting and tree construction
✅ Real-time visualization

📂 Installation & Setup

To run Tree-Visualizer locally, follow these steps:

Clone the repository:

git clone https://github.com/yourusername/Tree-Visualizer.git
cd Tree-Visualizer

Install dependencies: (if applicable)

npm install  # For Node.js projects

Run the project:

npm start  # or open index.html in a browser

🛠️ Usage

Enter an array of numbers in the input field.

Select the desired tree structure (Binary Tree, BST, Min Heap, Max Heap).

Click the "Visualize" button to generate the tree.

Observe how the numbers are organized and structured.
